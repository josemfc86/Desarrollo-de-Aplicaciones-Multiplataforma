Descripción de la aplicación "Programa_JavaFX";
Se ha desarrollado una aplicación denominada "Programa_JavaFX" en la cual se desarrollan dos
ventanas de dialogo con las herramientas de JavaFX y la aplicación "Scene Builder" la cual
nos permite crear estas interfaces de manera gráfica. La ventana principal es un AnchorPane
llamado "Gestión de Vehículos" donde mediante un item en la barra de menú o a través de un
botón podemos abrir una segunda ventana llamada "Coche Eléctrico" en la cual podemos
reservar un coche eléctrico al rellenar un conjunto de datos.
