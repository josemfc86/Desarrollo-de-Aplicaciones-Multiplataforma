En el proyecto Java (Tarea_DI07) nos encontramos con una aplicación de escritorio con entorno gráfico. La aplicación permite seleccionar un fichero de texto (extensión .txt) y buscar en su interior un texto. 
Para este proyecto se crea un instalable con NSIS, el cual tiene como objetivo, instalar y ejecutar la aplicación en Windows, e indica que se va a realizar la instalación de “Tarea_DI07” en el directorio de destino “C:\Program Files (x86)\Tarea_DI07”, así mismo se asigna un icono distinto al dado por defecto.
